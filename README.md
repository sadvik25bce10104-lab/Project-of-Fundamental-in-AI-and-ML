Smart Student Performance Predictor (AI/ML Project)
Project Overview
This project is a Machine Learning-based system that predicts a student's final performance based on
study habits and other factors. It demonstrates core AI/ML concepts using a simple regression model.
Features
1 Predicts student performance
2 Uses structured dataset
3 Implements Linear Regression
4 Beginner-friendly and easy to modify
Tech Stack
1 Python
2 Pandas
3 NumPy
4 Scikit-learn
5 Matplotlib (optional)
How to Run
1 Install Python 3
2 Install libraries: pandas, numpy, scikit-learn
3 Run the script using: python main.py
Input Parameters
1 Study Hours
2 Attendance (%)
3 Sleep Hours
4 Previous Scores
Output
1 Predicted Final Score
2 Model Error (MSE)
Future Improvements
1 Add GUI using Tkinter
2 Deploy as web application
3 Use real-world dataset
4 Apply advanced ML models
Conclusion
This project demonstrates how machine learning can be used to predict outcomes and assist in
decision-making.
